## 背词法
1. 约10个一组（一页），背第一遍，约5分钟。
2. 复习这一页，约30秒。（复习指盖住原文，试图回忆单词意思）
3. 重复步骤一和步骤二，约6~8组共计30分钟。
4. 复习此6~8组内容，约3分钟。
5. 重复以上步骤，每天约4个list（标准速度360词/天），12小时后复习一次。
6. 之后每隔1，2，4，7，15天后复习。
